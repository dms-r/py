# Peta Keterampilan Utama Data Science

## 1. Fondasi Matematis dan Statistik
- Sub-keterampilan:
  a) Matematika Dasar
     - Tingkat kesulitan: Sedang
     - Waktu penguasaan: 40-60 jam
     - Fokus: Aljabar linier, kalkulus, probabilitas

  b) Statistika Deskriptif dan Inferensial
     - Tingkat kesulitan: Tinggi
     - Waktu penguasaan: 60-80 jam
     - Fokus: Distribusi data, uji hipotesis, regresi

## 2. Pemrograman dan Komputasi
- Sub-keterampilan:
  a) Bahasa Pemrograman
     - Tingkat kesulitan: Sedang
     - Waktu penguasaan: 80-120 jam
     - Fokus: Python atau R, struktur data, algoritma

  b) Manipulasi Data
     - Tingkat kesulitan: Sedang
     - Waktu penguasaan: 40-60 jam
     - Fokus: Pandas, NumPy, transformasi data

## 3. Machine Learning
- Sub-keterampilan:
  a) Algoritma Dasar
     - Tingkat kesulitan: Tinggi
     - Waktu penguasaan: 100-150 jam
     - Fokus: Regresi, klasifikasi, clustering

  b) Model Lanjutan
     - Tingkat kesulitan: Sangat Tinggi
     - Waktu penguasaan: 150-200 jam
     - Fokus: Deep learning, neural network

## 4. Visualisasi dan Komunikasi Data
- Sub-keterampilan:
  a) Teknik Visualisasi
     - Tingkat kesulitan: Rendah-Sedang
     - Waktu penguasaan: 30-50 jam
     - Fokus: Matplotlib, Seaborn, Tableau

  b) Storytelling dengan Data
     - Tingkat kesulitan: Sedang
     - Waktu penguasaan: 40-60 jam
     - Fokus: Presentasi insight, dashboard

## 5. Alat dan Infrastruktur
- Sub-keterampilan:
  a) Cloud Computing
     - Tingkat kesulitan: Sedang
     - Waktu penguasaan: 50-70 jam
     - Fokus: AWS, Google Cloud, Azure

  b) Big Data
     - Tingkat kesulitan: Tinggi
     - Waktu penguasaan: 70-100 jam
     - Fokus: Spark, Hadoop, database distributed

# Sumber Belajar Paling Efisien

## Matematika dan Statistik
1. Coursera - "Statistics with Python" (University of Michigan)
2. Khan Academy - Modul Statistik Gratis
3. "Think Stats" - Buku Python untuk Statistik
- Tips: Praktikkan konsep dengan menggunakan dataset sederhana
- Urutan: Mulai dari konsep dasar → uji statistik → regresi

## Pemrograman
1. Codecademy - Kursus Python
2. DataCamp - Track Data Science
3. "Python for Data Analysis" - Buku Wes McKinney
- Tips: Kerjakan mini-proyek setelah setiap modul
- Urutan: Sintaks dasar → struktur data → manipulasi data

## Machine Learning
1. Coursera - "Machine Learning" Andrew Ng
2. Kaggle Learn
3. "Hands-On Machine Learning" - Buku Aurélien Géron
- Tips: Gunakan Scikit-learn untuk praktik
- Urutan: Algoritma klasik → deep learning → proyek kompleks

## Visualisasi
1. Udacity - Data Visualization Nanodegree
2. Plotly dokumentasi resmi
3. Tableau Desktop Learning
- Tips: Visualisasikan dataset pribadi secara rutin
- Urutan: Matplotlib → Seaborn → Plotly/Tableau

# Milestone Pencapaian

## 20 Jam Pertama
- Mampu menginstall Python
- Memahami konsep dasar statistik
- Membuat plot sederhana
- Minimal kompetensi: Mengolah dataset kecil dengan pandas

## 100 Jam
- Membangun model prediksi sederhana
- Membuat dashboard interaktif
- Mampu membersihkan dan mempersiapkan data

## 500 Jam
- Mengembangkan model machine learning kompleks
- Berkontribusi di proyek open-source
- Memecahkan masalah bisnis nyata

# Potensi Hambatan Umum

## Kesalahan Pemula
1. Terlalu fokus pada teori, kurang praktek
2. Tidak membersihkan data dengan benar
3. Memilih model yang terlalu kompleks
4. Mengabaikan interpretasi statistik

## Strategi Mengatasi Frustasi
- Bergabung dengan komunitas (Discord, Reddit)
- Fokus pada proyek nyata, bukan tutorial
- Dokumentasikan setiap proses belajar
- Gunakan metode Pomodoro untuk belajar

## Teknik Motivasi
- Ikuti kompetisi Kaggle
- Buat portofolio proyek pribadi
- Tetapkan target bulanan yang terukur
- Temukan mentor atau study group

# Rekomendasi Perjalanan Belajar

1. Bulan 1-2: Fondasi Matematika & Python
2. Bulan 3-4: Statistik & Manipulasi Data
3. Bulan 5-6: Machine Learning Dasar
4. Bulan 7-8: Proyek Kompleks & Spesialisasi
5. Bulan 9-12: Pengembangan Portofolio Profesional

Catatan Penting:
- Konsistensi lebih penting dari kecepatan
- Praktik langsung adalah kunci utama
- Selalu update dengan teknologi terbaru
- Jangan takut membuat kesalahan
